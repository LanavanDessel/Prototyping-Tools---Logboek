Webpagina’s maken met HTML en CSS

Lana van Dessel

Datum: 2025-10-09

Vandaag hebben we uitleg gekregen over hoe je gestructureerde en semantische webpagina’s kunt maken met HTML, en hoe je deze kunt opmaken met CSS. We moesten een website maken over het werk van de afgelopen weken (ons logboek). Ik vond het best ingewikkeld, omdat je veel tegelijk moet doen en sommige dingen lastig uitgelegd stonden. Toch vind ik dat het voor de eerste keer best goed ging. Het is nog niet perfect, want ik weet nog niet alles om het er helemaal professioneel uit te laten zien, maar ik heb wel veel geleerd.

Reflectie:

Ik wil nog beter leren hoe ik CSS kan gebruiken om mijn website mooier en overzichtelijker te maken. Ook wil ik oefenen met de structuur van HTML zodat ik beter begrijp hoe alles samenwerkt.
